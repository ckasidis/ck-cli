# ck-cli

CLI to bootstrap Monorepos application

## Features

- 🏠 PNPM Workspace
- 🚀 Turborepo
- 🌝 TypeScript
- ⚡️ Next.js
- 💎 Tailwind
- 🔒 Clerk
- 💿 Drizzle ORM
- ✅ ESLint
- ✨ Prettier
- ⚙️ Lint-staged
- 🔧 Commitlint
- 🐶 Husky

## Prerequisite

Have pnpm and ts-node installed globally

## Usage

execute src/index.ts

```
ts-node src/index.ts -h
```

to see all available options
